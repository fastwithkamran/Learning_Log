# CSS Learning Log

This repo contains my CSS Learning Codes and Exercises.

CSS (Cascading Style Sheets) is what makes a webpage look beautiful. It controls the layout, colors, fonts, spacing, and visual presentation of HTML elements.

## What I Learned

- Intro to CSS and how to link stylesheets
- Inline, internal and external CSS
- CSS selectors (class, id, attribute, pseudo)
- CSS box model (margin, padding, border)
- CSS fonts and colors
- CSS specificity and cascade rules
- CSS units and measurements (px, em, rem, %)
- CSS display property (block, inline, flex, grid)
- CSS shadows and outlines
- Styling lists with CSS
- CSS overflow property
- CSS position property (static, relative, absolute, fixed, sticky)
- CSS variables and custom properties
- CSS media queries for responsive design
- CSS float and clear
- More CSS selectors (nth-child, not, has, is)
- CSS Flexbox - complete guide
- CSS Grid - complete guide
- CSS transforms (rotate, scale, translate, skew)
- CSS transitions
- CSS animations and keyframes
- CSS object-fit and object-cover
- CSS filters
