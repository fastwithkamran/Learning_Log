# JavaScript Learning Log

This repo contains my JavaScript Learning Codes and Exercises.

JavaScript is the programming language of the web — it makes web pages interactive and dynamic. From DOM manipulation to async programming, it is the backbone of both frontend and backend (Node.js) development.

## What I Learned

- Variables, Data Types, Conditional Statements, Loops and Functions
- Strings and Arrays with built-in methods (map, filter, reduce)
- DOM & BOM — selecting, inserting and removing elements
- Events and Event Bubbling
- Callbacks, Promises and the Pyramid of Doom
- Async / Await operations and the Fetch API
- Error Handling using try, catch and finally
- Classes and Objects (OOP basics in JS)
