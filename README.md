# Fullstack Web Dev Learning Log

Welcome to my Fullstack Web Development learning journey! This repository documents everything I've learned — from the very basics of HTML to building full-stack applications with React and Node.js.

## 📁 What's Inside

| Folder | Topics Covered
|--------|---------------|
| `html_codes/` | Structure, forms, semantic tags, media
| `css_codes/` | Selectors, flexbox, grid, animations
| `javascript_codes/` | Core JS, DOM, async/await, events
| `react_codes/` | Hooks, state, props, Fetch API
| `node_codes/` | Modules, file handling, HTTP server
| `express_codes/` | REST APIs, routing, middleware
| `fastapi_codes/` | Python backend, REST APIs
